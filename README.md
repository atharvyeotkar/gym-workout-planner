# 🏋️ Gym Workout Planner

A beginner-friendly Python mini project that helps users check their daily workout routine and calculate BMI.

## Features

- 📅 Today's Workout
- 📖 Full Week Workout Plan
- ⚖️ BMI Calculator
- ❌ Invalid Input Handling

## Technologies

- Python

## How to Run

```bash
python gym_workout_planner.py
```
